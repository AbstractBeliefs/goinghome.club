Weebs~
======

Development
-----------

Clone this repo::

    $ git clone https://github.com/Wishf/goinghome.club.git

Start a new virtualenv::

    $ cd goinghome.club
    $ virtualenv weeb

Activate the virtualenv::

    $ source weeb/bin/activate

Install the requirements::

    $ pip install -r requirements.txt

Run it::

    $ python app.py
